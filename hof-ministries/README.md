## House of Faith Website

This is the official website for House of Faith Ministries USA Inc., a welcoming church located in the Bronx, NY.

### Features

- Modern, mobile-friendly design
- Homepage with church introduction and navigation
- About, Ministries, Events, Give, and Contact pages
- Easy to update and extend

### Getting Started

To run the website locally:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Customization

Replace placeholder content and images in the `/src/app` directory as needed for your church.
